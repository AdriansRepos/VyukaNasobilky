Program pro výuku násobilky pro děti.



Hlavní menu dělí program na malou a velkou násobilku.

Každá volba má podmenu, kde se příklady dají procvičovat buď za sebou tak, jak jdou (tabulka), 

nebo náhodně generované.

Při dosažení aktuálního stupně výuky (například násobilka po číslo 4 včetně) je možné

místo výsledku zadat volbu 0 a potvrdit ji klávesou enter. Tím se dostaneme do možností podmenu.

Při opětovné volbě příkladů tak, jak jdou (Tabulkové) program opět začne vypisovat příklady od začátku.



Pro úplné ukončení aplikace stačí zadat místo výsledku nulu, tu potvrdit klávesou enter, poté se dostaneme do 

podmenu, kde se stiskem klávesy s nulou dostaneme do hlavního menu.



Ukončení programu provedeme stiskem klávesy s číslem 3.
